mac 下安装fastadmin

  • PHP
  • 2018-07-22

1.安装方式

通过终端命令安装：

  • 1.克隆FastAdmin到你本地git clone https://gitee.com/karson/fastadmin.git
    1. 进入目录 cd fastadmin
    1. 下载前端插件依赖包 bower install
    1. 下载PHP依赖包composer install
    1. 一键创建数据库并导入数据php think install -u 数据库用户名 -p 数据库密码
    1. 添加虚拟主机并绑定到fastadmin/public目录

补充：

  • 1.安装bower
  • 2.安装composer

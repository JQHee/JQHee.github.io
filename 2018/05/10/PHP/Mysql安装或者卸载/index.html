3.修改root密码
首先执行下面命令为了能够修改任意的密码
mysql> FLUSH PRIVILEGES;

之后执行修改密码的SQL语句,这里的qsd19001008可以替换你自己想要修改的密码

mysql> SET PASSWORD FOR root@'localhost' = PASSWORD('newpassword');

最后刷新
mysql>FLUSH PRIVILEGES;
